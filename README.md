# 国际化插件
改进自[wujushan/AndroidLocalizationer](https://github.com/wujushan/AndroidLocalizationer)
使用方式不变：

1. 本地安装插件
2. 然后申请对应翻译的key填入
3. 选中string.xml右键选中"Convert to other languages"，选择对应的语言
